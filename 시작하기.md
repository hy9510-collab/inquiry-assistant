# 다른 컴퓨터에서 질의서 비서 켜기

이 폴더(`질의서/`)만 있으면 어느 컴퓨터에서든 비서를 쓸 수 있습니다.
무거운 설치폴더(`node_modules`)는 빼고 옮겨도 됩니다 — 아래 1회 설치로 자동 복원됩니다.

## 새 컴퓨터에서 한 번만 준비 (3단계)

1. **Node.js 설치** — https://nodejs.org 에서 LTS 버전 설치 (한글파일 변환 엔진이 사용).
2. **Claude Code 설치 후 이 폴더 열기** — 비서의 "두뇌"는 Claude입니다. 이 폴더의 `CLAUDE.md`를 읽고 비서로 동작합니다.
   - Claude Code가 없으면, `CLAUDE.md` 내용을 Claude(claude.ai) 대화창에 붙여넣고 시작해도 됩니다.
3. **변환 엔진 설치 (1회)** — 터미널에서:
   ```bash
   cd kordoc-local
   npm install
   ```
   (인터넷 연결 필요. 끝나면 다시 안 해도 됨.)

## 평소 사용법

- `자료/`에 자료를 넣고, Claude에게 **유형만 말한다**(예: "결산 질의서 써줘").
- Claude가 요약 → 쟁점 → 질의서 작성 → 한글파일(.hwpx)까지 만들어 `출력/`에 넣는다.

## 작성·변환 작업실로 쓰기 (무료·로그인 불필요)

명령어 대신 **페이지에서 클릭으로** 질의서·보고서 한글파일을 만들 수 있습니다.

```bash
cd kordoc-local
node server.mjs
```

- 켜지면 안내되는 주소(예: `http://localhost:3939`)를 브라우저로 엽니다.
- ① **종류**를 고른다(질의서 / 현안·정책 검토보고서 / 활동·결과 보고서).
- ② Claude가 써 준 원고를 붙여넣거나 파일(.md/.txt)로 불러온다. 비어 있으면 **[빈 양식 넣기]**로 뼈대부터 시작.
- ③ **[한글파일 만들기]** → 다운로드.
- 끄려면 그 터미널 창에서 `Ctrl+C`. (인터넷·계정 필요 없음, 내 컴퓨터 안에서만 동작)

## 페이지에서 AI로 초안 만들기 (무료 Gemini · 선택)

Claude 없이 **페이지 안에서** 바로 초안을 뽑고 싶을 때 씁니다. 구글 Gemini 무료 키만 한 번 넣으면 됩니다(카드 등록 불필요).

**무료 키 넣기 (한 번만)**
1. 구글 계정으로 https://aistudio.google.com/apikey 접속.
2. **[Create API key / API 키 만들기]** 클릭 → 만들어진 키를 복사.
3. 메모장을 열고 **키만** 붙여넣기(따옴표·공백 없이) → `kordoc-local` 폴더에 **`gemini_key.txt`** 로 저장.
   - 저장할 때 파일 형식을 '모든 파일'로 하고 이름을 정확히 `gemini_key.txt` 로 (끝에 `.txt.txt`가 붙지 않도록).
4. 서버를 껐다(`Ctrl+C`) 다시 켠다(`node server.mjs`). 터미널에 **`AI 자동작성: 준비됨`**이 보이면 완료.

**쓰는 법**
- 종류를 고르고, **[AI로 초안 만들기]** 옆 칸에 주제/지시를 적은 뒤 버튼을 누른다(보통 10~30초).
- 원고칸에 자료를 먼저 붙여넣으면 그 자료를 참고해서 씁니다.
- 나온 초안은 **반드시 내용을 확인·수정**한 뒤 [한글파일 만들기]로 변환한다(AI는 사실·수치를 틀리게 쓸 수 있음).

**주의 (보안)**
- `gemini_key.txt`는 **비밀번호와 같습니다.** 드라이브·깃허브·메신저에 올리지 마세요(내 컴퓨터에만 둡니다).
- [AI로 초안 만들기]를 누르면 입력한 주제·자료가 **구글 서버로 전송**되어 처리됩니다. 민감한 자료는 넣지 않는 것이 안전합니다.
- 시간이 지나 모델이 안 되면 `server.mjs`의 `GEMINI_MODEL` 값만 최신 모델명으로 바꾸면 됩니다.

## 한글파일 변환 명령 (참고)

```bash
cd kordoc-local
node build.mjs "../출력/<원고>.md" "../출력/<결과>.hwpx"
```
