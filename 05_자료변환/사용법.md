# 자료변환 시스템 사용법

자료(파일·링크·메모)를 넣으면 **Gemini가 정리**해서 성격에 맞는 문서
(**hwpx · 엑셀 · ppt · pdf · docx**)로 만들어 주는 로컬 웹앱입니다.

## 처음 한 번만 (준비)

1. **Gemini API 키 발급** — https://aistudio.google.com/apikey
   구글 로그인 → **Create API key** → 키 복사 (무료 등급으로 충분)
2. 이 폴더의 **`설정.txt`** 를 열어 `GEMINI_API_KEY=` 뒤에 키를 붙여넣고 저장
   ```
   GEMINI_API_KEY=AIza...복사한키...
   ```

## 매번 쓰는 법

1. 명령창(또는 탐색기)에서 프로젝트 폴더로 이동해 실행:
   ```
   python 자료변환_웹.py
   ```
   > `python` 별칭이 안 먹으면 전체 경로:
   > `%LOCALAPPDATA%\Programs\Python\Python312\python.exe 자료변환_웹.py`
2. 브라우저가 자동으로 열립니다 (http://127.0.0.1:8765)
3. **자료 넣기** — 파일 드래그 / 링크 붙여넣기 / 메모 입력 (여러 개 가능)
4. **출력 형식** 선택 (자동 = 성격에 맞게 Gemini가 결정)
5. **정리해서 만들기** → 결과 파일을 내려받기 (원본은 `출력/` 폴더에도 저장됨)

## 넣을 수 있는 자료
- 문서: hwpx · xlsx · pptx · docx · pdf · txt · md · csv · json
- 링크: 웹페이지 주소 (본문 자동 수집)
- 미디어: 이미지 · 음성 · 동영상 (19MB 이하 — Gemini가 직접 보고/듣고 정리)

## 나오는 형식
| 형식 | 언제 |
|---|---|
| hwpx | 보고서·공문·회의 요약 |
| xlsx | 표·명단·수치 데이터 |
| pptx | 발표·브리핑 자료 |
| pdf  | 배포·인쇄용 |
| docx | 워드 문서 |

## 참고
- 모든 자료·결과물은 이 PC 안에만 저장됩니다(깃 제외). 자료 원문은 Gemini로 전송되어 정리됩니다.
- 민원인 등 개인정보는 정리 단계에서 이니셜·유형만 남기고 비식별 처리하도록 지시돼 있습니다.
